     \hypertarget{md_public_2_muse_2assets_2vendor_2flatpickr_2_r_e_a_d_m_e_autotoc_md159}{}\doxysection{\texorpdfstring{Motivation}{Motivation}}\label{md_public_2_muse_2assets_2vendor_2flatpickr_2_r_e_a_d_m_e_autotoc_md159}
Almost every large SPA or project involves date and time input. Browser\textquotesingle{}s native implementations of those are inconsistent and limited in functionality. Most other libraries require you to pull in heavy dependencies like j\+Query, Bootstrap, and moment.\+js. I wanted something that was good-\/looking out of the box, dependency-\/free, powerful, and extensible.

Feature overview\+:


\begin{DoxyItemize}
\item Dependency-\/free (no bloated bundles)
\item Simple, polished UX
\item Date + time input
\item Range selections
\item Ability to select multiple dates
\item Can be used as just a time picker
\item Display dates in a human-\/friendly format
\item Easily disable specific dates, date ranges, or any date using arbitrary logic
\item Week numbers
\item 51 locales
\item 8 colorful themes (incl. dark and material)
\item Numerous plugins
\item Libraries available for React, Angular, Vue, Ember, and more
\end{DoxyItemize}
